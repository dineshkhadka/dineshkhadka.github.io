Fast

Junkiree starts up and completely loads in just a few seconds. Thanks to its minimal dependencies.

Simple

Junkiree is simple and easy to use. Everything is laid out right where it should be.

clean

Junkiree is presented in a clean and simple layout. A lot of focus and effort has been given to the app design.

Fast

Junkiree is built to be fast. Junkiree doen't import a ton of library for its featues. It only uses jQuery as the main library. Nepali Date converter and jQuery Knob are bundled in to provide additional features to the app. More speed increases will be added along the way. All of them will be posted in github.

simple

Junkiree is extremly easy to use. Junkiree doesn't provide any fancy bloat, we have focused on providing only the important stuff. The interface is easy for anyone to use. The app provides easy and effortless usage without any obstruction. Plus, you are free to fiddle with the code.

Clean

Junkiree is free of clutter . The primary goal of junkiree was to try to display loadshedding with a interface that doesn't cause your eye to itch. I've seen a lot of Nepalese apps whose UI have made me cringe (not that I'm implying mine is any better). We will continu to work on this in the future.

About Junkiree

Junkiree was made due to lack of any loadshedding apps for the desktop platform or at least any usable ones. While the app has been open sourced in order to share the code and also because I wanted people to suggest improvements to the code and the design. The app has been made using modern technologies and methods. Everything is simple and open.
